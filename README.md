# aboutJumpMacro

```
究極的にシンプルなジャンプマクロを実装する。
最大の問題は、サブルーチン。

１．ラベル宣言
２．キーウェイト
３．ウェイト
４．変数格納
５．イフジャンプ

１．ラベル宣言
ラベルは、ジャンプする為の指標である。
#label

２．キーウェイト
キー呼び出しを一度呼びだし、キーを取得する。キーが押されない限り、制御は戻らない。
k>
//check $k

３．ウェイト
時間を指定し、その時間待機する。時間がたたない限り、制御は戻らない。
w>100
//100ms

４．変数格納
可能な限り、広域を汚さない。
$wk=xyz

５．イフジャンプ
条件式をかけなければ、時間がかかる。
{1}>>>#label1 //常にジャンプ
{$k==='A'}>>>#labal2 //キーがAであったら。

```

```
//ループ

#start
$count=0
#start.loop

k>
$count+=1
{1}>>>#start.loop

//jump back
#echo

k>
{1}>>>###


```
